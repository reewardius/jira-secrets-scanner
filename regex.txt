# ============================================================================
# BALANCED Secret Scanning Patterns
# Format: Name:::Regex:::GroupIndex
# ============================================================================
# Improvements:
# - Generic patterns now require assignment operator (= or :)
# - Generic patterns require quotes around values
# - Password/Secret require exact variable names (password, secret)
# - JWT/Bearer tokens kept (useful for real credential detection)
# - Minimum length increased for generic patterns
# ============================================================================
